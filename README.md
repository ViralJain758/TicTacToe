# TicTacToe Game 

A clean, responsive **Tic Tac Toe** game built using **HTML, CSS, and Vanilla JavaScript**.  
This project focuses on core DOM manipulation, game logic, and simple UI feedback without using any external libraries.

- Reset functionality to restart the game
- Responsive layout for smaller screens
- Minimal, modern UI inspired by clean design systems

---

##  Game Logic Overview

- The game alternates turns between **X** and **O**
- Each cell can be clicked only once
- After every move:
  - The game checks for a winning combination
  - If a player wins, the game stops further input
  - A victory message and animation are shown
- The reset button clears the board and restarts the game
